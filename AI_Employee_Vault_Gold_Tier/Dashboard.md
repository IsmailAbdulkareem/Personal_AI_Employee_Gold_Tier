## Quick Actions

### Start Watchers
```bash
# Start all watchers via orchestrator
python orchestrator.py run

# Or start individually
python watchers/gmail_watcher.py
python watchers/whatsapp_watcher.py
python watchers/linkedin_watcher.py
```

### Check Status
```bash
# Orchestrator status
python orchestrator.py status

# Individual watcher status
python watchers/gmail_watcher.py status
python watchers/whatsapp_watcher.py status
python watchers/linkedin_watcher.py status
```

### Process Tasks
```bash
# Process pending tasks (auto-generates plans + approvals)
python orchestrator.py process

# Check approvals
python scripts/approval_workflow_enhanced.py check

# Execute approved actions
python scripts/approval_workflow_enhanced.py execute

# Update dashboard
python scripts/dashboard_updater.py
```

### Gold Tier Actions
```bash
# Start Ralph Wiggum Loop for autonomous tasks
python scripts/ralph_wiggum_loop.py "Process all pending tasks" --vault AI_Employee_Vault_Gold_Tier

# Generate weekly audit report
python scripts/audit_logger.py report --vault AI_Employee_Vault_Gold_Tier

# Query audit logs
python scripts/audit_logger.py query --start-date 2026-03-01

# Generate CEO briefing (when implemented)
python scripts/ceo_briefing_generator.py --vault AI_Employee_Vault_Gold_Tier
```
